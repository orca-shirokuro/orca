FAN KIT追加方法

■ フリー配布素材を追加
1. 画像を assets/fankit/frames/ にアップロード
2. data/rings.json に1件追加

■ コラボ配布素材を追加
画像を assets/fankit/collab/ に入れて data/collab_downloads.json に追加

■ 限定コレクションを追加
画像を assets/special/ に入れて data/special_collection.json に追加
download は false のまま

■ ゼト×ルカギャラリーを追加
画像を assets/gallery/zetoluka/ に入れて data/zetoluka_gallery.json に追加
保存対策あり・ダウンロード不可

■ ちーぺん。ギャラリーを追加
画像を assets/gallery/chipen/ に入れて data/chipen_extra.json に追加
